치지직을 더 편하게 쓰도록 검색·플레이어·레이아웃 기능을 한데 모은 브라우저 확장 프로그램입니다. 채널의 `동영상` 탭과 `클립` 탭에 검색 도구를 추가하고, 별도 팝업 창에서는 스트리머 닉네임으로 다른 채널의 콘텐츠도 검색할 수 있습니다. 라이브와 다시보기 플레이어에서는 오디오 믹서·비디오 필터로 방송별 소리와 화면을 보정하고, 실시간 따라잡기로 라이브 지연을 줄일 수 있습니다. 헤더와 사이드바를 정리하고, 사이드바를 숨겨도 헤더 바로가기로 주요 메뉴에 접근할 수 있습니다. 확장 아이콘의 기능 설정에서 필요한 기능만 카테고리별로 켜고 끌 수 있습니다.

---

## 주요 기능

### 다시보기 검색

- 제목, 태그, 카테고리 검색을 지원합니다.
- 시작일/종료일과 빠른 기간 선택을 제공합니다.
- 달력에서 `2023년`부터 현재 년도까지 년/월을 빠르게 선택할 수 있습니다.
- 길이 필터, `다시보기`, `업로드`, `시청 중`, `시청안함` 유형 필터를 제공합니다.
- 최신순, 오래된순, 조회순, 댓글 많은순, 라이브 시청순 정렬을 지원합니다.
- 시청 중인 영상은 카드 하단 진행 바로 시청 위치를 표시합니다.
- 검색 결과를 표시할 때 기존 치지직 페이지네이션은 숨기고, 초기화하면 원래대로 복구합니다.

### 다시보기 타임스탬프

- 다시보기 댓글에 포함된 `1:23`, `1:23:45`, `(1:23)` 형식의 타임스탬프를 플레이어 슬라이더에 마커로 표시합니다.
- 댓글 타임스탬프 버튼으로 마커 목록을 열고, 항목을 클릭해 해당 재생 위치로 이동할 수 있습니다.
- 타임스탬프 마커에 마우스를 올리면 플레이어 미리보기 아래에 댓글 내용을 표시합니다.
- 다시보기 재생바를 탐색할 때 미리보기 시간 아래에 방송 당시 실제 시각을 함께 표시합니다.
- 타임스탬프 목록을 열면 현재 재생 시간이 속한 항목을 강조하고, 재생 위치가 다음 구간으로 넘어가면 강조 항목도 함께 이동합니다.
- 타임스탬프 목록 패널에서 재생바 마커 표시를 켜고 끌 수 있습니다.
- 댓글 타임스탬프 버튼 우클릭 메뉴에서 타임스탬프 기능 전체를 켜고 끌 수 있습니다.
- `bestComments`와 설명이 있는 댓글을 우선 표시하고, `3초` 이내의 비슷한 타임스탬프는 하나로 묶습니다.
- 한 댓글 안에 여러 타임스탬프가 있거나 괄호 안에 추가 타임스탬프가 있는 경우도 각각 추출합니다.
- 타임스탬프가 없으면 버튼을 흐리게 표시하고 팝업이 열리지 않도록 처리합니다.

### 오디오 믹서

- 치지직 라이브와 다시보기 플레이어 하단에 오디오 믹서 버튼을 추가합니다.
- 기본, 저챗·라디오, 게임 방송, 야외방송, 노래 방송, 클래식·재즈, 영화·드라마, 애니, 스포츠, ASMR 프리셋을 제공합니다.
- 고급 슬라이더와 전문가 모드에서 게인, EQ, 컴프레서, 리미터, 노멀라이저 목표 레벨을 조정할 수 있습니다.
- 오디오 믹서가 켜져 있을 때 플레이어 하단 믹서 버튼 옆에서 마스터 게인을 바로 조절할 수 있습니다.
- 치지직 음량 슬라이더와 오디오 믹서 마스터 게인 슬라이더를 조절할 때 현재 값을 % 툴팁으로 표시합니다.
- 치지직 기본 음량 슬라이더가 접힌 상태에서도 볼륨 컨트롤 위에 마우스를 올리거나 조작하면 % 툴팁을 안정적으로 표시합니다.
- 확장 설정에서 `오디오 믹서 항상 켜기`를 켜면 페이지 첫 조작 이후 믹서를 자동으로 활성화합니다.
- 프리셋을 수정한 뒤 현재 설정을 커스텀 프리셋으로 저장하고, 커스텀 프리셋을 수정하거나 삭제할 수 있습니다.
- 커스텀 프리셋은 JSON으로 내보내거나, 공유받은 JSON을 붙여넣어 불러올 수 있습니다.
- 커스텀 프리셋을 `기본` 프리셋 대체값으로 등록해 기본 칩에서 바로 적용할 수 있습니다.
- 프리셋을 수정한 상태에서는 적용 중이던 프리셋 이름을 표시하고, 초기화 버튼으로 수정 전 프리셋 값으로 되돌릴 수 있습니다.
- 프리셋별 노멀라이저 목표 레벨과 리미터 값을 적용해 방송 유형에 맞는 음량 보정을 제공합니다.
- 믹서 버튼의 상태 표시 dot으로 활성/비활성 상태를 구분합니다.
- 플레이어가 미니플레이어로 전환되거나 재렌더되어도 오디오 믹서 연결을 유지하거나 새 영상 요소에 다시 연결합니다.
- 같은 영상의 오디오 소스를 다른 확장 프로그램이 이미 사용 중이면 믹서를 끄고 충돌 안내를 표시합니다.
- 스트림 정보 버튼으로 해상도, FPS, 비트레이트, 코덱, 하드웨어 가속 가능 여부, 라이브 레이턴시, 오디오 정보를 확인할 수 있습니다.
- 플레이어 하단의 `탭 음소거` 버튼으로 치지직 플레이어 음소거와 별개로 브라우저 탭 전체를 음소거하거나 해제할 수 있습니다.
- 라이브 상세의 시청자 수/스트리밍 시간 영역에 마우스를 올리면 라이브 시작일을 툴팁으로 확인할 수 있습니다.
- 라이브와 다시보기 진입 시 치지직 플레이어의 `넓은 화면` 모드를 자동으로 켤 수 있습니다.
- 라이브 플레이어에서 설정한 간격의 `되감기`와 `앞으로` 버튼으로 seekable 구간 안에서 이동할 수 있고, 플레이어 위에서는 좌우 방향키로도 같은 이동을 사용할 수 있습니다.
- 라이브 플레이어 컨트롤 위에 되감기 가능 구간 바를 표시하고, 바를 드래그해 원하는 지점으로 이동할 수 있습니다.
- 라이브 지연이 커지면 `실시간 따라잡기` 버튼을 활성화하고, 클릭 시 일시적으로 배속을 높여 라이브 엣지에 가까워지도록 보정합니다.
- `실시간 따라잡기` 버튼 우클릭 메뉴에서 자동 따라잡기를 켜면 라이브 지연이 커질 때 자동으로 따라잡기를 실행합니다.
- 확장 설정에서 실시간 따라잡기 민감도를 `낮음`, `보통`, `높음` 프리셋이나 `커스텀`(시작/목표 지연 직접 입력)으로 조정할 수 있습니다.
- 라디오 모드처럼 오디오 전용 스트림인 경우 스트림 정보 패널에서 비디오 정보를 비우고 오디오 전용 상태와 오디오 비트레이트를 우선 표시합니다.

### 비디오 필터

- 치지직 라이브와 다시보기 플레이어 하단에 비디오 필터 버튼을 추가합니다.
- 밝기, 노출, 대비, 채도, 색온도, 색조, 감마, 선명도, 그림자, 하이라이트를 조정할 수 있습니다.
- 원본, FPS 게임, 롤·AOS, 게임 일반, 공포 게임, 야외방송, 스포츠, 먹방·쿡방, 캠방송, 버츄얼, 야간 시청, 시네마틱 프리셋을 제공합니다.
- CSS filter와 SVG filter를 함께 사용해 기본 색감 보정뿐 아니라 감마, 선명도, 그림자, 하이라이트 보정을 적용합니다.
- 선명도가 높은 필터를 적용할 때 하드웨어 가속 안내를 표시하고, 프레임 드롭이 감지되면 선명도를 일시적으로 낮추는 자동 조절 옵션을 제공합니다.
- 확장 설정에서 `비디오 필터 항상 켜기`를 켜면 채널 진입 시 필터를 자동으로 활성화하고, 채널별로 직접 끈 상태는 유지합니다.
- 프리셋을 수정한 뒤 현재 설정을 커스텀 프리셋으로 저장하고, 수정 전 프리셋 값으로 초기화할 수 있습니다.
- 커스텀 비디오 필터 프리셋은 JSON으로 내보내거나, 공유받은 JSON을 붙여넣어 불러올 수 있습니다.

### 채팅 정리

- 채팅창의 치즈/후원 랭킹, 진행 중인 미션, 승부예측 패널을 숨길 수 있습니다.
- 채팅창 좌측 경계를 드래그해 너비를 조절하고, 채팅창을 플레이어 왼쪽에 배치할 수 있습니다.
- 채팅창을 자동으로 숨기고, 마우스를 화면 좌우 가장자리 근처에 가져가면 슬라이드로 다시 표시할 수 있습니다.
- 다시보기 채팅창도 너비 조절과 왼쪽 배치를 지원하며, 조절한 너비에 맞춰 영상 정보 영역과 미니플레이어 폭을 보정합니다.
- 채팅창 위 배너 광고가 조절한 채팅 폭을 벗어나지 않도록 배너와 내부 광고 영역의 폭을 함께 보정합니다.
- 채팅 폰트, 이모티콘, 배지 크기를 `80%`부터 `200%`까지 조절할 수 있습니다.
- 각 채팅 앞에 `HH:MM` 형식의 보낸 시각을 표시할 수 있습니다.
- 클린봇 또는 블라인드로 가려진 채팅을 원문으로 복원해 흐린 이탤릭 텍스트로 구분해 표시할 수 있습니다.
- `배지 모아 챗`이 같은 채팅 기능을 제어 중이면 충돌을 피하기 위해 해당 기능을 자동으로 양보합니다.

### 통나무파워

- 라이브 채팅 입력창 부근에 현재 채널의 보유 통나무파워를 표시할 수 있습니다.
- 시청 중 받을 수 있는 통나무파워 보상을 자동으로 획득할 수 있습니다.
- 1시간 시청 보상까지 남은 시간을 표시하고, 보상 팝업이 뜨면 자동 획득과 토스트 알림을 처리할 수 있습니다.
- 시청 적립 상태는 백그라운드 알람으로 1분마다 추적해 탭이 숨겨지거나 치지직 내부 화면이 전환되어도 적립 진행 표시를 이어갑니다.
- 한 채널에서 통나무파워 적립이 확인되면 다른 채널의 적립 표시와 1시간 보상 타이머를 정리해 중복 표시를 방지합니다.
- 라이브가 종료되면 통나무파워 적립 추적과 1시간 보상 타이머를 정리합니다.

### 레이아웃 정리

- 헤더의 `스튜디오` 버튼을 숨겨 헤더를 깔끔하게 정리할 수 있습니다.
- 헤더의 게임, e스포츠, 스포츠, 엔터+ 주제 탭을 숨길 수 있습니다.
- 헤더를 자동으로 숨기고 화면 맨 위에 마우스를 올렸을 때만 다시 표시할 수 있습니다.
- 왼쪽 사이드바 전체를 숨기고, 비워진 공간만큼 콘텐츠 영역을 넓힐 수 있습니다.
- 사이드바를 화면 오른쪽에 배치하고, 콘텐츠 영역 여백도 오른쪽 기준으로 맞출 수 있습니다.
- 사이드바의 `전체 방송`, `인기 클립`, `카테고리`, `편성표`, `팔로잉`, `치즈팜` 메뉴와 `팔로우`, `인기 카테고리`, `방송 일정`, `파트너`, `서비스 바로가기` 섹션을 항목별로 숨길 수 있습니다.
- 팔로잉 섹션에서 방송 중이 아닌 오프라인 채널만 가릴 수 있습니다.
- 사이드바와 헤더 주제 탭을 함께 숨긴 경우, 팔로우 라이브 채널을 헤더의 주제 탭 자리에서 캐러셀로 확인할 수 있습니다.
- 헤더 팔로우 툴팁은 방송 제목, 카테고리, 시청자 수, 인증/업적 배지를 표시하며 자동 숨김 헤더가 닫혀 있을 때는 갱신을 보류합니다.
- 헤더에 표시되는 팔로우 라이브 채널 수를 직접 선택할 수 있으며, 좁은 화면에서는 검색창과 겹치지 않도록 표시 개수를 자동으로 줄입니다.
- 사이드바를 숨겨도 헤더에 `전체 방송`, `인기 클립`, `카테고리`, `편성표`, `팔로잉`, `치즈팜` 바로가기 아이콘을 표시할 수 있으며, 표시할 항목을 직접 선택할 수 있습니다.
- 헤더 바로가기는 페이지 전체를 새로고침하지 않고 치지직 자체 화면 전환으로 이동하며, 마우스를 올리면 아래에 메뉴 이름 툴팁을 표시합니다.
- 사이드바가 펼쳐진 상태에서 팔로우 채널 목록을 주기적으로 새로고침해 라이브 상태를 갱신할 수 있습니다(끔/30초/60초/커스텀 3-600초). 10초 이하로 설정하면 호출 제한을 피하기 위해 설정값과 설정값-10초 랜덤 간격을 번갈아 호출합니다.
- 사이드바 팔로잉 섹션의 `더보기`를 누르면 접기 상태가 될 때까지 추가 목록을 자동으로 펼치고, 갱신 후에도 펼침 상태를 복원합니다.
- 사이드바와 헤더의 팔로잉 라이브 채널에 마우스를 올리면 라이브 영상 미리보기 패널을 표시하고, 패널 크기 조절과 고정 표시를 지원합니다.
- 팔로잉 라이브 미리보기는 설정한 자동 종료 시간이 지나면 닫히며, 같은 채널에 계속 호버 중일 때 즉시 다시 열리지 않도록 처리합니다.
- 팔로잉 라이브 미리보기는 음소거 고정, 썸네일만 보기, 헤더 폰트 크기 조절(75~175%)을 지원합니다.
- 팔로잉 라이브 미리보기 영상 로딩이 실패하면 패널을 닫지 않고 썸네일로 대체 표시합니다.
- 채널 홈 탭 영역에 현재 채널의 라이브 페이지로 이동하는 바로가기 버튼을 표시할 수 있고, 라이브/오프라인 상태에 따라 버튼 스타일을 구분합니다.
- 라이브 탐색 카드 미리보기 위에서 우클릭으로 음소거를 토글하고, 마우스 휠로 미리보기 음량을 조절할 수 있습니다.
- 다시보기의 `영상 더보기` 영역을 영상 아래로 옮기거나 숨길 수 있으며, 숨길 때 자동 재생 스위치가 켜져 있으면 먼저 끕니다.
- 채팅창이 없는 다시보기에서 `영상 더보기` 배치/숨김을 적용할 때 영상 영역 오른쪽에 빈 공간이 남지 않도록 플레이어 폭을 보정합니다.

### 기능 설정

- 확장 아이콘을 누르면 좌측 카테고리 탭(`전체`, `플레이어`, `다시보기`, `검색`, `레이아웃`, `사이드바`, `미리보기`, `채팅`, `통나무파워`, `클립`)으로 구분된 기능 설정 팝업을 열 수 있습니다.
- 오디오 믹서, 비디오 필터, 실시간 따라잡기, 라이브 되감기, 스트림 정보, 탭 음소거, 댓글 타임스탬프, 다시보기/클립 검색, 사이드바 표시 여부를 개별적으로 숨기거나 다시 표시할 수 있습니다.
- 넓은 화면 자동 적용, 라이브 되감기·앞으로 이동 간격, 라이브 되감기 바 표시, 다시보기 영상 더보기 아래 배치, 다시보기 영상 더보기 숨김을 설정할 수 있습니다.
- 오디오 믹서 항상 켜기, 비디오 필터 항상 켜기, 헤더 스튜디오 버튼, 헤더 주제 탭, 헤더 자동 숨김, 사이드바 위치, 사이드바 항목/섹션, 팔로잉 오프라인 채널, 헤더 바로가기 항목 표시 여부, 헤더 팔로우 표시 개수를 설정할 수 있습니다.
- 팔로잉 라이브 미리보기와 라이브 카드 미리보기 음소거·음량 조작을 켜고 끌 수 있으며, 팔로잉 미리보기 자동 종료 시간을 `30초`, `1분`, `2분`, `3분`, `5분` 중에서 선택할 수 있습니다.
- 팔로잉 라이브 미리보기의 음소거 여부, 썸네일 전용 표시, 헤더 폰트 크기를 설정할 수 있습니다.
- 채팅 정리, 채팅창 자동 숨김, 통나무파워 관련 기능을 카테고리별로 설정할 수 있습니다.
- 채널 홈의 라이브 바로가기 버튼 표시 여부와 탭 영역 내 배치 위치를 설정할 수 있습니다.
- 다시보기 탐색 미리보기의 실제 방송 시각 표시를 끄고 켤 수 있습니다.
- 실시간 따라잡기 민감도를 `낮음`, `보통`, `높음` 프리셋(시작/목표 지연 시간 명시)이나 직접 입력하는 `커스텀`으로 설정할 수 있습니다.
- 팔로우 채널 자동 갱신 주기를 프리셋 또는 커스텀(3~600초)으로 설정할 수 있습니다.
- 클립 페이지의 `클릭하여 라이브 시청` 버튼은 기본적으로 숨기고, 설정에서 다시 표시할 수 있습니다.
- 라이트 모드와 다크 모드를 지원하며, 우측 패널만 스크롤되고 팝업을 열 때마다 `전체` 탭에서 시작합니다.

### 클립 검색

- 클립 제목과 카테고리 검색을 지원합니다.
- 치지직의 `전체`, `24시간`, `7일`, `30일`, `최신순`, `조회순` 조건을 따라 수집합니다.
- 최신순, 오래된순, 조회순, 좋아요순 정렬을 지원합니다.
- 치지직 클립 카드와 유사한 세로형 카드 UI로 결과를 표시합니다.
- 검색 결과 카드는 원본 목록처럼 화면 폭에 따라 4열, 3열, 2열, 1열로 반응형 전환됩니다.
- 썸네일 비율에 따라 세로형/가로형 blur 배경 표현을 구분합니다.
- 재생 수, 좋아요 수, 등록일, 재생 시간, 카테고리 pill을 함께 보여줍니다.
- 19세 연령 제한 클립은 치지직 원본 카드와 유사한 제한 표시를 적용합니다.
- 클립 수집 중에는 기존 결과를 점진적으로 표시하고, 진행 문구와 애니메이션으로 상태를 안내합니다.
- 카테고리 pill을 클릭하면 해당 카테고리만 필터링하고, `Ctrl + 클릭` 또는 macOS의 `Cmd + 클릭`으로 치지직 카테고리 페이지를 엽니다.

### 내가 만든 클립 검색

- 치지직 스튜디오의 `내가 만든 클립` 탭에서 클립 제목과 스트리머명을 검색할 수 있습니다.
- 시작일/종료일로 만든 날짜 범위를 필터링할 수 있습니다.
- 스트리머별 분류와 최신순, 오래된순, 조회순, 댓글 많은순 정렬을 지원합니다.
- 검색 결과에서도 관리, 더보기, 링크 복사, iframe 복사, 삭제 동작을 사용할 수 있습니다.
- 원본 페이지에서 클립이 삭제되거나 검색 결과에서 삭제한 경우 캐시를 갱신해 삭제된 클립이 다시 나타나지 않도록 처리합니다.
- 삭제 요청 실패 후에도 실제 목록에서 클립이 사라졌는지 잠시 확인하고, 삭제된 것으로 확인되면 성공 처리합니다.

### 팝업 검색

- 채널 페이지의 `팝업` 버튼으로 넓은 별도 창에서 검색할 수 있습니다.
- 스트리머 닉네임으로 다른 채널의 다시보기와 클립을 검색할 수 있습니다.
- 검색 후보가 여러 명이면 사용자가 직접 채널을 선택할 수 있습니다.
- 후보 목록에는 채널 이미지, 채널명, 공식 인증 마크가 표시되며, 채널 ID는 필요할 때만 보거나 복사할 수 있습니다.
- 공식 인증 마크 이미지를 불러오지 못하면 텍스트 배지로 대체합니다.
- 클립 팝업의 초기 준비 상태와 검색 중 상태에는 스켈레톤 UI와 animated SVG를 표시합니다.
- 라이트 모드와 다크 모드를 지원합니다.

### 안정성과 캐시

- API 요청과 전체 수집은 background service worker에서 처리합니다.
- 다시보기와 클립은 페이지당 최대 `50개`씩 요청합니다.
- 댓글 타임스탬프는 치지직 댓글 API에서 최대 `5페이지`까지 확인하고, 결과를 `30분` 동안 메모리에 캐시합니다.
- 다시보기는 나머지 페이지를 최대 `3개`까지 병렬 요청합니다.
- 클립은 커서 기반 페이지네이션이므로 순차적으로 수집합니다.
- 클립 좋아요순 정렬에 필요한 좋아요 수는 별도 반응 API로 조회하며, 지연이나 실패가 반복되면 검색 완료를 우선합니다.
- 서로 다른 전체 수집 작업은 최대 `2개`까지만 동시에 실행하고 나머지는 대기합니다.
- 동일한 목록을 여러 화면에서 동시에 요청하면 진행 중인 수집 작업을 공유합니다.
- 검색 중에는 content의 `검색` 버튼과 팝업 새로고침 버튼이 `검색 중지` 동작으로 바뀝니다.
- 수집 결과는 메모리와 `chrome.storage.local`에 캐시하며, 기본 TTL은 `1시간`입니다.
- 큰 클립 목록은 chunk 단위로 저장하고, content와 팝업 모두 chunk 캐시를 복원할 수 있습니다.
- 클립이 중간에 삭제될 수 있으므로 누락 항목은 한 번 더 확인한 뒤 삭제된 항목으로 처리합니다.
- 치지직 스튜디오의 `내가 만든 클립` 삭제 XHR을 감지해 원본 페이지와 검색 캐시를 함께 갱신합니다.
- 치지직 페이지의 CSS 모듈 클래스명이 변경되어도 동영상/클립 링크와 현재 DOM 구조를 함께 확인해 검색 도구와 결과 목록을 복구합니다.
- 기능 설정의 표시/숨김 상태와 실시간 따라잡기 민감도는 `chrome.storage.local`에 저장하고 열린 치지직 탭에 즉시 반영합니다.
- 오디오 믹서 설정은 현재 라이브/다시보기 기준으로 저장하고, 커스텀 프리셋은 전체 플레이어에서 공유합니다.
- 비디오 필터 설정은 현재 라이브/다시보기 기준으로 저장하고, 커스텀 프리셋은 전체 플레이어에서 공유합니다.
- 통나무파워 시청 적립 상태는 `chrome.alarms`와 `chrome.storage.session`을 사용해 백그라운드에서 채널별로 추적합니다.
- 확장 프로그램 업데이트 시 열려 있는 치지직과 치지직 스튜디오 탭에 새로고침 안내 배너를 표시합니다.

## 사용 방법

### 채널 페이지에서 검색

1. `https://chzzk.naver.com/{channelId}/videos` 또는 `https://chzzk.naver.com/{channelId}/clips`로 이동합니다.
2. 페이지에 추가된 치즈 플래터 검색 도구에서 검색어와 필터를 설정합니다.
3. `검색` 버튼을 눌러 목록을 불러옵니다.
4. 목록을 가져온 뒤에는 검색어, 기간, 유형, 정렬 조건을 바꿔 즉시 다시 필터링할 수 있습니다.

### 다시보기 타임스탬프

1. 치지직 다시보기 재생 페이지(`/video/{videoNo}`)로 이동합니다.
2. 댓글에 포함된 `1:23`, `1:23:45`, `(1:23)` 같은 타임스탬프를 자동으로 수집합니다.
3. 수집된 타임스탬프는 플레이어 슬라이더에 마커로 표시되며, 마커를 클릭하면 해당 시점으로 이동합니다.
4. 마커에 마우스를 올리면 플레이어 미리보기 아래에 댓글 내용이 표시됩니다.
5. 재생바를 탐색하면 미리보기 시간 아래에 방송 당시 실제 시각이 함께 표시됩니다.
6. 플레이어 하단의 댓글 타임스탬프 버튼을 누르면 전체 타임스탬프 목록을 팝업으로 확인할 수 있습니다.
7. 패널 상단 스위치로 재생바 마커 표시 여부를 켜고 끌 수 있습니다.
8. 댓글 타임스탬프 버튼을 우클릭하면 타임스탬프 기능 전체를 켜거나 끌 수 있습니다.

비슷한 타임스탬프는 `3초` 이내 기준으로 하나로 묶고, `bestComments`와 설명이 있는 댓글을 우선 표시합니다. 타임스탬프 목록은 현재 재생 구간을 강조하며, 타임스탬프가 없으면 버튼은 흐리게 표시되고 팝업이 열리지 않습니다.

### 오디오 믹서 사용

1. 치지직 라이브 또는 다시보기 재생 페이지로 이동합니다.
2. 플레이어 하단의 오디오 믹서 버튼을 눌러 패널을 엽니다.
3. 방송 유형에 맞는 프리셋을 선택하거나, 고급 슬라이더/전문가 모드에서 직접 값을 조정합니다.
4. 믹서가 켜진 상태에서는 플레이어 하단 믹서 버튼 옆 슬라이더로 마스터 게인을 빠르게 조정할 수 있습니다.
5. 프리셋을 수정한 설정은 `프리셋 추가`로 커스텀 프리셋에 저장할 수 있습니다.
6. 수정한 값은 `초기화` 버튼으로 수정 전 프리셋 값으로 되돌릴 수 있습니다.
7. 커스텀 탭의 `내보내기`/`불러오기`로 프리셋 JSON을 복사하거나 붙여넣을 수 있습니다.
8. 스트림 정보 버튼을 눌러 현재 재생 중인 영상/오디오 정보를 확인할 수 있습니다.
9. 라이브 지연이 큰 경우 `실시간 따라잡기` 버튼으로 재생 속도를 잠시 높여 지연을 줄일 수 있습니다.
10. `실시간 따라잡기` 버튼을 우클릭하면 자동 따라잡기를 켜거나 끌 수 있습니다.

### 비디오 필터 사용

1. 치지직 라이브 또는 다시보기 재생 페이지로 이동합니다.
2. 플레이어 하단의 비디오 필터 버튼을 눌러 패널을 엽니다.
3. 방송 유형에 맞는 화면 프리셋을 선택하거나, 슬라이더로 밝기와 색감을 직접 조정합니다.
4. 수정한 설정은 `프리셋 추가`로 커스텀 프리셋에 저장할 수 있습니다.
5. 선명도 값이 높아 영상이 버벅이면 `선명도 자동 조절`을 켜 프레임 드롭에 따라 선명도를 일시적으로 낮출 수 있습니다.
6. 커스텀 탭의 `내보내기`/`불러오기`로 비디오 필터 프리셋 JSON을 복사하거나 붙여넣을 수 있습니다.

### 기능 설정 사용

1. 브라우저 툴바에서 치즈 플래터 확장 아이콘을 누릅니다.
2. 좌측 카테고리 탭(`전체`, `플레이어`, `검색`, `레이아웃`, `사이드바`, `클립`)에서 원하는 분류를 선택합니다.
3. 사용하지 않을 기능의 스위치를 켜면 해당 버튼이나 UI가 숨겨집니다.
4. 실시간 따라잡기 민감도에서 `낮음`, `보통`, `높음` 또는 `커스텀`을 선택합니다.
5. 사이드바를 숨겼다면 헤더 바로가기에 표시할 메뉴를 선택할 수 있습니다.
6. 클립 페이지의 `클릭하여 라이브 시청` 버튼은 기본적으로 숨겨지며, 필요하면 설정에서 다시 표시할 수 있습니다.

### 팝업에서 다른 채널 검색

1. 채널 페이지의 치즈 플래터 검색 도구에서 `팝업` 버튼을 누릅니다.
2. 팝업 상단의 스트리머 입력란에 닉네임을 입력합니다.
3. 검색 결과가 여러 명이면 후보 목록에서 원하는 채널을 선택합니다.
4. 해당 채널의 다시보기 또는 클립 목록을 검색합니다.

## 검색 문법

검색어는 일반 단어뿐 아니라 `OR`, `AND`, 제외 조건, 괄호를 조합할 수 있습니다.

| 기능              | 예시                                                          | 설명                                        |
| ----------------- | ------------------------------------------------------------- | ------------------------------------------- |
| 일반 검색         | `대회 결승`                                                   | 두 단어를 모두 포함한 결과를 찾습니다.      |
| OR 검색           | `대회 \| 결승`, `대회 OR 결승`                                | 하나 이상의 단어를 포함한 결과를 찾습니다.  |
| AND 검색          | `대회 AND 결승`                                               | 모든 단어를 포함한 결과를 찾습니다.         |
| 제외 검색         | `대회 -예선`                                                  | 특정 단어를 포함한 결과를 제외합니다.       |
| 괄호 조합         | `(대회 \| 이벤트) 결승`                                       | 여러 조건을 묶어서 검색합니다.              |
| 태그 검색         | `#게임`                                                       | 다시보기 태그에서 검색합니다.               |
| 카테고리 검색     | `@메이플`, `category:메이플`, `cat:메이플`, `카테고리:메이플` | 카테고리에서만 검색합니다.                  |
| 띄어쓰기 카테고리 | `category:"메이플스토리 월드"`, `@"메이플스토리 월드"`        | 빈칸이 포함된 카테고리를 정확히 검색합니다. |

다시보기는 제목, 태그, 카테고리를 대상으로 검색합니다. 클립은 제목과 카테고리를 대상으로 검색합니다.

## 참고 사항

- 치지직의 페이지 구조나 API가 변경되면 일부 기능이 동작하지 않을 수 있습니다.
- 전체 목록을 처음 불러오는 시간은 채널의 다시보기 또는 클립 개수와 네트워크 상태에 따라 달라집니다.
- 클립 목록은 커서 기반으로 수집하므로 다시보기보다 최초 로딩 시간이 길 수 있습니다.
- 댓글 타임스탬프 기능은 댓글 API 응답 구조와 플레이어 슬라이더 DOM 구조에 의존합니다.
- 오디오 믹서 기능은 브라우저 Web Audio API와 치지직 플레이어 DOM 구조에 의존합니다.
- 팔로잉 라이브 미리보기는 치지직 라이브 상세 API와 HLS 재생 환경에 의존하며, 브라우저나 네트워크 상태에 따라 로딩 시간이 달라질 수 있습니다.
- 비디오 필터 기능은 브라우저 CSS/SVG 필터와 치지직 플레이어 DOM 구조에 의존합니다.
- 선명도 필터는 브라우저와 기기 성능에 따라 부담이 클 수 있으며, 높은 선명도에서는 하드웨어 가속을 켜는 것이 좋습니다.
- 브라우저 정책상 같은 영상의 오디오는 하나의 `MediaElementSource`만 만들 수 있으므로, 다른 확장이 이미 오디오 그래프를 사용 중이면 오디오 믹서를 켤 수 없습니다.
- 실시간 따라잡기 기능은 치지직 라이브 플레이어의 레이턴시 정보와 브라우저 재생 속도 제어에 의존합니다.
- 클립 라이브 이동 버튼 숨김은 치지직 클립 페이지와 `m.naver.com/shorts` iframe DOM 구조에 의존합니다.
- 본 확장 프로그램은 치지직과 관련이 없으며, 네이버, 치지직, CHZZK은 NAVER의 등록상표입니다.
